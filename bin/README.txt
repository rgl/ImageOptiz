ImageOptiz is a GUI front end for image optimization tools.

The following tools are used for optimizing PNG files:

	* optipng 0.6.4 (http://optipng.sourceforge.net/)
	* advpng 1.15 (http://advancemame.sourceforge.net/comp-readme.html)
	* pngcrush 1.7.13 (http://pmt.sourceforge.net/pngcrush/)
	* pngout 2010-06-09 (http://advsys.net/ken/utils.htm)
	  NB: You need to manually download and copy the executable
	      into the "tools" subdirectory.

		  This tool is highly recommended.


The following tools are used for optimizing JPEG files:

	* jpegoptim 1.2.2 (http://freshmeat.net/projects/jpegoptim/)
	* jpegtran (http://jpegclub.org/jpegtran/)


ImageOptiz was inspired by http://code.google.com/p/imageoptim/
